Privacy Policy

This Privacy Policy explains what data TrapHub Bot may store and how it is used.

1. Data Collected

TrapHub Bot may store limited Discord-related data required for bot features.

  • Discord user ID
  • XP and level data
  • Economy balance, such as coins
  • Warnings and moderation records
  • Ticket transcripts, if the ticket system is used
  • Giveaway participation data

2. How Data Is Used

The collected data is used only to provide server features such as moderation, verification, tickets, economy, levels, giveaways and server management.

3. Data Sharing

TrapHub Bot does not sell, trade or share user data with third parties. Data is used only inside the Discord server where the bot operates.

4. Data Storage

Data may be stored in local bot files or databases used by the bot. Access to this data is limited to the bot owner or authorized server administration.

5. Data Removal

Users may request removal of their stored data by contacting the TrapHub server administration on Discord.

6. Security

Reasonable steps are taken to keep stored data secure. However, no system can be guaranteed to be completely secure.

7. Contact

If you have questions about this Privacy Policy, contact the TrapHub server administration through Discord.
